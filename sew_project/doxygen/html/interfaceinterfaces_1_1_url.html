◆ getFileName()

String interfaces.Url.getFileName ( )
Returns
Returns the filename (with extension) of the main.url path. If the main.url contains no filename, a empty string is returned. Never returns null. A filename is present in the main.url, if the last segment contains a name with at least one dot.

Implemented in main.url.UrlClass.

◆ getFragment()
